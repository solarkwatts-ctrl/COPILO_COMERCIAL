# Copiloto Comercial IA V13 Final

## Instalación sobre la base actual
1. Ejecute `supabase/05_V13_SAAS_MULTIEMPRESA_FINAL.sql` una sola vez.
2. Mantenga las variables de `.env.local` y Vercel.
3. Reinicie `npm run dev`.

## Incluye
- Superadministrador multiempresa.
- Empresa Demo separada de empresas reales.
- Selector de empresa activa.
- Creación y suspensión de empresas.
- Usuarios, roles, sucursales y zonas por empresa.
- Dashboards y menús por rol.
- Cargas, auditoría y backups aislados por `empresa_id`.
- IA filtrada por empresa y rol.
